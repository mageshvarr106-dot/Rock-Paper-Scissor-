🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.
Play against the computer, track your score, and enable auto-play mode.

⸻

🚀 Live Demo

👉 https://mageshvarr106-dot.github.io/Rock-Paper-Scissor-/

⸻

✨ Features
	•	🎮 Rock, Paper, Scissors gameplay
	•	🤖 Auto Play mode
	•	💾 Score saved using LocalStorage
	•	🔄 Reset Score button
	•	📱 Mobile responsive design
	•	🎨 Clean UI with emoji icons

⸻

🛠️ Built With
	•	HTML
	•	CSS
	•	JavaScript (Vanilla JS)
	•	Local Storage API
